{% trans "Statistical Methods: Mean vs Median Analysis" %}

{% trans "Traditional Mean Analysis:" %}
  • • {% trans "Uses arithmetic average of all grade scores" %}
  • • {% trans "Sensitive to extreme values (outliers)" %}
  • • {% trans "Can be distorted by a few very high or low scores" %}
  • • {% trans "Standard method in most educational research" %}
  • • {% trans "Shown as solid lines in chart" %}
{% trans "Example:" %} {% trans "Grades [60, 65, 70, 75, 100] → Mean = 74" %}
{% trans "Robust Median Analysis:" %}
  • • {% trans "Uses middle value when scores are sorted" %}
  • • {% trans "Resistant to extreme values (outlier-proof)" %}
  • • {% trans "Represents typical student experience" %}
  • • {% trans "Includes quartile ranges (Q1-Q3) for spread analysis" %}
  • • {% trans "Shown as dashed lines in chart" %}
{% trans "Same Example:" %} {% trans "Grades [60, 65, 70, 75, 100] → Median = 70" %}
{% trans "Why Both Matter:" %} {% trans "Comparing mean and median reveals whether extreme scores are distorting the analysis. If they differ significantly, outliers are present and the median provides a more reliable picture of typical student performance." %}
